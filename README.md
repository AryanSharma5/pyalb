pyalb is a simple layer 4 http load balancer.

Installation
------------

Install from PyPI::

    $ pip install pyalb

or

    $ git clone

    $ cd pyalb

    $ python -m pip install .

or install in editable version

    $ git clone

    $ cd pyalb

    $ python -m pip install -r requirements-dev.txt
