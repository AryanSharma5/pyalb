pyalb is a simple http load balancer. It supports following routing algorithms

    - Round Robbin
    - Random Selection

Installation
------------

Install from PyPI::

    $ pip install pyalb

or::

    $ git clone
    $ cd pyalb
    $ python -m pip install .

or install in editable version::

    $ git clone
    $ cd pyalb
    $ python -m pip install -r requirements-dev.txt
